🌐 How to Create a Website on GitHub

GitHub Pages is a free hosting platform that allows developers to build and deploy professional websites directly from a GitHub repository.

1. Create a GitHub Account

  • Go to https://github.com
  • Sign up and verify your email

2. Create a New Repository

  • Click New Repository
  • Name: username.github.io
  • Set to Public
  • Initialize with README

3. Clone Repository

git clone https://github.com/username/username.github.io.git
cd username.github.io

4. Create Website Files

index.html

<!DOCTYPE html>
<html>
<head>
<title>My Professional Website</title>
</head>
<body>
<h1>Welcome to My Website</h1>
<p>Hosted on GitHub Pages</p>
</body>
</html>

style.css (optional)

body {
  font-family: Arial;
  background: #f4f4f4;
}

5. Add Images

images/
 └── profile.jpg

Use in HTML:

<img src="images/profile.jpg" alt="Profile Image">

6. Commit and Push

git add .
git commit -m "Initial setup"
git push origin main

7. Enable GitHub Pages

  • Go to Settings → Pages
  • Select branch: main
  • Folder: /root

8. Templates & Themes

  • Use GitHub Themes
  • Use HTML templates (HTML5 UP)

9. Continuous Updates

git pull origin main

File Structure

username.github.io/
│── index.html
│── style.css
│── images/
│    └── profile.jpg
└── README.md

Tips

  • Use consistent design
  • Optimize images
  • Add portfolio & contact sections
  • Use custom domain (optional)

Types

Applications
