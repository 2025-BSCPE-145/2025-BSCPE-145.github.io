🚀 My First GitHub Pages Website

A fully custom-built static website created using HTML and CSS, hosted on GitHub Pages without any templates or frameworks.

📌 Project Overview

This project represents my first step into real web development. I built a personal website from scratch using only HTML and CSS, and deployed it using GitHub Pages. The goal was to understand how websites are structured, styled, and published on the internet.

🎯 Objectives

  • Learn how HTML structures a webpage
  • Understand CSS styling and layout systems
  • Practice building multi-page websites
  • Deploy a live website using GitHub Pages

🛠️ Technologies Used

  • HTML5 – Structure and content
  • CSS3 – Styling and layout design
  • Git & GitHub – Version control and hosting

⚙️ Development Process

I started by creating a simple index.html file and gradually expanded it into a multi-page website. I added navigation between pages like Home, Blog, About, and Contact. Then I styled everything using CSS to make it visually appealing and responsive for different screen sizes.

After completing the design, I uploaded the project to GitHub, configured GitHub Pages, and deployed it successfully. This helped me understand the full lifecycle of a real-world website.

💡 Key Learnings

  • How websites are structured using semantic HTML
  • How CSS Grid and Flexbox create layouts
  • How to design clean UI without templates
  • How GitHub Pages hosting works
