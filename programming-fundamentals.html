📘 Programming Fundamentals

Beginner Level • 5 min read

💡 What is Programming?

Programming is the process of giving instructions to a computer to perform tasks.

📦 Variables

Variables store data values.

let name = "Noori";
let age = 20;

🔁 Loops

Loops repeat tasks efficiently.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

⚡ Conditions

Used for decision making.

if (age > 18) {
  console.log("Adult");
}
