How to Make a Website on GitHub Pages

GitHub Pages allows you to host websites for free directly from a GitHub repository. Follow these steps to create your own website:

Step 1: Create a GitHub account at github.com and verify your email.

Step 2: Create a new repository named yourusername.github.io (important).

Step 3: Add your website files:
✔ index.html (main page)
✔ style.css (design file)

Step 4: Upload files to GitHub using “Add file → Upload files”.

Step 5: Go to Settings → Pages → Select “Deploy from branch (main)”.

Step 6: Your website will be live at:
https://yourusername.github.io
