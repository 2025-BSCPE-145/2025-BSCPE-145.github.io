Database Systems

🔹 What is a Database?

A database is an organized collection of data that can be easily accessed, managed, and updated. It is used to store information efficiently.

🔹 Types of Databases

1. Relational Databases

Use tables with rows and columns (MySQL, SQL Server).

2. NoSQL Databases

Store unstructured data (MongoDB).

3. Distributed Databases

Data is stored across multiple systems.

🔹 DBMS (Database Management System)

A DBMS is software used to manage databases (e.g., MySQL, Oracle).

🔹 SQL Basics

SELECT * FROM students;

INSERT INTO students VALUES (1, 'Ali');

DELETE FROM students WHERE id = 1;

🔹 Importance of Databases

  • Efficient data storage 📊
  • Fast retrieval ⚡
  • Data security 🔐
  • Used in real-world systems (banks, apps)
