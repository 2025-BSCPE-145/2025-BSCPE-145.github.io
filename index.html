🌐 GitHub Pages Website Deployment Project

A step-by-step demonstration of how to build and deploy a personal website using GitHub Pages. This project covers repository setup, file structuring, and live deployment.

GitHub HTML CSS Hosting

🔧 Process Overview

  • Create GitHub Account: Register at github.com and verify email
  • Create Repository: Name it yourusername.github.io
  • Add Files: Upload index.html and optional style.css
  • Enable Pages: Go to Settings → Pages → Deploy from main branch
  • Launch: Website goes live on GitHub Pages URL

💻 Sample index.html




<!DOCTYPE html>
<html lang="en">
<head>
  <title>My Portfolio</title>
</head>

<body>
  <h1>Hello, I am Noor 👩💻</h1>
  <p>Welcome to my GitHub website!</p>
</body>
</html>

📊 Database Management Systems (DBMS)

A Database Management System (DBMS) is software used to store, manage, and organize data efficiently in structured form.

A DBMS acts as an interface between users and databases, allowing secure, efficient, and structured data handling in applications like banking, education, and e-commerce.

🔑 Core Functions of DBMS

  • Data storage and retrieval
  • Data security and access control
  • Transaction management
  • Concurrency control (multiple users)

🏗️ DBMS Architecture Concepts

  • Physical Level – how data is stored
  • Logical Level – what data is stored
  • View Level – user-specific view

📚 Reference

Learn more about DBMS fundamentals: DBMS Full Guide (Contabo)
