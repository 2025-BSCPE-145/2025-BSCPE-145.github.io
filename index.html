How to Make a Website on GitHub (GitHub Pages)

🧩 Step 1: Create a GitHub Account

  • Go to: (link unavailable)
  • Sign up (if you don’t already have an account)
  • Verify your email

📁 Step 2: Create a New Repository

This is your website “project folder”.

  • Click: 👉 New Repository
  • Fill:
    • Repository name: (link unavailable) (VERY IMPORTANT ⚠️ must match exactly)
    • Set to Public
    • Tick Add README (optional)
  • Then click: 👉 Create repository

💻 Step 3: Add Your Website Files

You need at least:

  1. index.html (main page)
    2. <!DOCTYPE html>
<html>
<head>
<title>My Portfolio</title>
</head>
<body>
<h1>Hello, I am Noor 👩💻</h1>
<p>Welcome to my GitHub website!</p>
</body>
</html>
  2. (Optional) style.css
    3. body {
font-family: Arial;
text-align: center;
background: #f4f4f4;
}

⬆️ Step 4: Upload Files to GitHub

Inside your repository:

  • Click Add file
  • Choose Upload files
  • Upload:
    • index.html
    • style.css (if you have it)
  • Then click: 👉 Commit changes

🌍 Step 5: Turn on GitHub Pages

Go to:

  • Settings
  • Scroll to Pages

Then:

  • Source: Deploy from branch
  • Branch: main
  • Folder: /root
  • Click: 👉 Save

🎉 Step 6: Open Your Website

After 1–2 minutes, your site will be live at:
