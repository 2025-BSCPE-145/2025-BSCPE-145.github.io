Building My First Responsive Website

In this lab task, our teacher asked us to make a personal website that looks good on both phone and computer.

What I Did

I used HTML for the structure and CSS for styling. I made a nice header with navigation and a hero section that introduces me.

What I Learned

  • How to use Flexbox to arrange items nicely
  • Making the website adjust automatically on mobile (responsive design)
  • Using CSS variables to easily change colors
  • Adding smooth hover effects on cards

This was my first complete project. It was challenging at first but now I feel more confident with basic web development.
