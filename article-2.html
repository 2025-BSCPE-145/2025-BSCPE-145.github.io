📌 Introduction

Programming fundamentals form the backbone of all software development. These concepts allow developers to build efficient, scalable, and reliable systems.

🧠 Core Concepts

1. Variables
age = 20
name = "Noor"
isStudent = True
2. Conditional Statement
if age >= 18:
    print("Adult")
else:
    print("Minor")
3. Loop Example
for i in range(5):
    print(i)

🔄 Flowchart (Even or Odd)

Start → Input n → Check (n % 2 == 0)
        ↓Yes              ↓No
     Print Even      Print Odd
             → End

🔁 Loop Flowchart

Start → i = 1 → Check (i ≤ 5)
        ↓Yes              ↓No
     Print i           End
        ↓
     i = i + 1 → repeat

🧠 Memory Model
