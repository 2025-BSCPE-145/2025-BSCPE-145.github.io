Python Programming

🔹 Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in web development, artificial intelligence, data science, and automation.

🔹 Features of Python

1. Easy Syntax

Python uses simple English-like syntax, making it beginner-friendly.

2. Interpreted Language

No need to compile; code runs line by line.

3. Large Standard Library

Includes modules for web, data, AI, and more.

4. Cross-Platform

Runs on Windows, Linux, and Mac.

🔹 Basic Concepts

for and while loops
