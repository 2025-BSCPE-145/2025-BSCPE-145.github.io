🌐 How to Make a Website on GitHub (GitHub Pages)

🧩 Step 1: Create a GitHub Account

  • Go to: https://github.com
  • Sign up (if you don’t already have an account)
  • Verify your email

📁 Step 2: Create a New Repository

  • Click 👉 New Repository
  • Repository name: yourusername.github.io (must match exactly)
  • Set to Public
  • Tick Add README (optional)
  • Click 👉 Create repository

💻 Step 3: Add Website Files

index.html

<!DOCTYPE html>
<html>
<head>
  <title>My Portfolio</title>
</head>
<body>
  <h1>Hello, I am Noor 👩‍💻</h1>
  <p>Welcome to my GitHub website!</p>
</body>
</html>

style.css (optional)

body {
  font-family: Arial;
  text-align: center;
  background: #f4f4f4;
}

⬆️ Step 4: Upload Files

  • Add file → Upload files
  • Upload index.html + style.css
  • Click 👉 Commit changes

🌍 Step 5: Enable GitHub Pages

  • Go to Settings
  • Scroll to Pages
  • Source: Deploy from branch
  • Branch: main
  • Folder: /root
  • Click 👉 Save

🎉 Step 6: Open Website

Your site will be live at:

https://yourusername.github.io

📊 Database Systems: A Comprehensive Guide

In the digital era, data is often referred to as the “new oil.” Every interaction generates information that must be stored and processed efficiently. Database Systems provide a structured and secure way to manage this data in modern applications.

What is a Database System?

A Database System consists of a database (organized data), a DBMS (software to manage it), and users/applications interacting with it.

Core Components

  • Hardware: Servers and storage devices
  • Software: DBMS like MySQL, PostgreSQL
  • Data: Structured information
  • Users: DBAs, developers, end-users
  • Procedures: Rules for managing data

Types of Database Systems

  • Relational (RDBMS): Tables (MySQL, PostgreSQL)
  • NoSQL: Flexible data (MongoDB)
  • Hierarchical: Tree structure
  • Network: Linked records
  • Object-Oriented: Data as objects

Advantages

  • Data accuracy and integrity
  • Security and access control
  • Efficient querying
  • Backup and recovery
  • Scalability

Real-World Applications

  • Banking systems
  • E-commerce platforms
  • Healthcare records
  • Social media applications
  • Education systems

💻 Programming Fundamentals – Beginner to Pro Guide

Programming is the process of writing instructions that a computer can execute. It is the foundation of all modern software systems.

Why Learn Programming?

  • Enhances problem-solving skills
  • Helps build real-world applications
  • Foundation for advanced technologies (AI, Web, Cybersecurity)

Core Concepts

Variables & Data Types

  • Integer, Float, String, Boolean

Operators

  • Arithmetic (+, -, *, /)
  • Comparison (==, >, <)
  • Logical (and, or, not)

Control Structures

if age >= 18:
    print("Adult")
else:
    print("Minor")

Loops

for i in range(5):
    print(i)

Functions

def greet(name):
    return "Hello " + name

Data Structures

  • List
  • Dictionary
  • Set
  • Tuple

Learning Path

  1. Start with one language (Python recommended)
  2. Practice small programs
  3. Learn data structures
  4. Build projects

Conclusion

Programming fundamentals are the building blocks of all software development. Mastering them enables you to create powerful applications and solve real-world problems efficiently.
