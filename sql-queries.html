Introduction to Database Systems

Date: April 2026

In this post, I am sharing basic SQL queries that I learned in my Database Systems course at university.

1. Creating a Table

CREATE TABLE students (
    student_id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    email VARCHAR(100),
    enrollment_date DATE,
    gpa DECIMAL(3,2)
);

2. Inserting Records

INSERT INTO students (student_id, first_name, last_name, email, gpa)
VALUES 
    (101, 'Ali', 'Khan', 'ali@example.com', 3.75),
    (102, 'Sara', 'Ahmed', 'sara@example.com', 3.90),
    (103, 'Noor', 'Fatima', 'noor@example.com', 3.85);

3. Retrieving Data (SELECT)

-- Get all data
SELECT * FROM students;

-- Get specific columns
SELECT first_name, last_name, gpa FROM students;

-- Filter students with high GPA
SELECT first_name, last_name 
FROM students 
WHERE gpa >= 3.8;

-- Sort by GPA (highest first)
SELECT first_name, last_name, gpa 
FROM students 
ORDER BY gpa DESC;

4. Updating and Deleting

-- Update GPA
UPDATE students 
SET gpa = 3.95 
WHERE student_id = 102;

-- Delete a record
DELETE FROM students 
WHERE student_id = 103;

5. Simple JOIN Query

SELECT 
    s.first_name, 
    s.last_name, 
    c.course_name
FROM students s
JOIN enrollments e ON s.student_id = e.student_id
JOIN courses c ON e.course_id = c.course_id;

What I Learned

  • SQL is used to create, read, update, and delete data (CRUD).
  • Primary Key ensures unique identification of records.
  • Always filter data using WHERE clause for better performance.
  • Use ORDER BY to sort results.
